Tell us the use cases of null and undefined.

Null and Undefined are two primitive values in JavaScript

  • Null: Represents the absence of a variable's value or a property has no value.
  • Undefined: Represents the absence of a variable ,property or function argument.

Discuss the scope of var, let, and const.

Var,let and const are used to declare variables (introduced in ES6 - let and const) with different scope and behaviors

  • Var: Variables declared with var can be accessed all over the function. Regardless of where it is declared, it is hoisted to the top of the function.Can be redeclared within the same function
  • Let: Variables declared with let are only accessible within the block they are declared in, but can't be redeclared within the same block.No hoisting take place here.
  • Const: Const variable have a block scope just like 'let'. No hoisting ,reassignment and redeclaration take place in 'const' variables .

What do you mean by REST API?

Representational State of Resource or REST API is an architectural style for designing networked apps. Based on the idea of resources which are identified by URIs(Uniform Resource Identifiers), which can be manipulated using a fixed set of operations

  • Key characteristics of REST API: Resource-based, client-server architecture, stateless,cacheable, uniform interface, layered system
  • REST APIs typically use HTTP methods to interact with resources: GET,POST,PUT,DELETE
